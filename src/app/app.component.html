### Creation Operators * **of:** https://rxjs.dev/api/index/function/of - **Usage:** create new observable from plain value - **Notes:** also works with more than one value * **from:** https://rxjs.dev/api/index/function/from - **Usage:** create new observable from promise - **Notes:** also works with arrays, iterables, scheduler, ...
```typescript // of of('of') .subscribe(console.log); ``` ```typescript // from const p = new Promise(resolve => setTimeout(() => resolve('from'), 1000) ); from(p) .subscribe(console.log); ```
